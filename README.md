## To run server
Required: python ( 3.10+ better), pip, virtualenv ( optional )

Download code
- git clone https://github.com/mphstar/gesture-recognition-kiosk-server.git

To install:
1. pip install -r requirements.txt
2. python GestureWithImage.py
